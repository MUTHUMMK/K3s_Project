K3s Infrastructure + CI/CD Deployment Project

This project demonstrates a complete **DevOps workflow** including: - Infrastructure provisioning using Terraform - Kubernetes cluster setup using K3s - Containerization using Docker - CI/CD automation using GitHub Actions - Cloud infrastructure on AWS The pipeline automatically: 1. Creates a K3s server 2. Builds Docker images 3. Pushes images to Docker Hub 4. Deploys the application into Kubernetes
